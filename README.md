# 🚗 New Car Purchase Prediction

This project is a simple **Machine Learning Web Application** built using **Flask**.  
It predicts whether a person will purchase a car based on **Age** and **Salary**.

---

## Project Overview

- Build a Machine Learning model for prediction  
- Deploy the model using Flask  
- Provide a simple web interface for user input  

---

## Technologies Used

- Python  
- Flask  
- NumPy  
- Pandas  
- Scikit-learn  
- HTML  

---

## Install Dependencies
  pip install flask numpy pandas scikit-learn

---

## Train Model
  Run your training script to generate:
   - model.pkl

---

## Run Application
- python app.py

---

## Open in your local Browser
- http://127.0.0.1:5000/

---

## Input Features

- Age  
- Salary  

---

## Output

- Will Purchase  
- Will Not Purchase  

---

## Model Used

- Decision Tree  

---

## How It Works

1. User enters Age and Salary  
2. Data is sent to Flask backend  
3. Model predicts result  
4. Output is displayed on webpage
